Les balises obligatoires

Tout document HTML doit obligatoirement comporter au minimum les balises suivantes :

Balise Description
<!DOCTYPE> L'inclusion du DOCTYPE dans un document HTML assure que le navigateur interprétera la version HTML ainsi déclarée. En HTML5 le doctype à déclarer est : <!DOCTYPE html>.
<head> Pour les informations d'entête du document HTML. Elles ne sont pas affichées mais interprétées par le navigateur.
<body> Partie principale du document HTML. Le contenu de cette partie est affiché à l'écran par le navigateur.
